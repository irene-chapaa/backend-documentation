# Confirm Phone API

**Endpoint:**

```
POST {{Url}}/auth/confirmPhone
```

**Authorization:** Bearer Token

```
{{Token}}
```

**Request Body**:

```json
{
  "phone": "+254746764503",
  "otp": "809169"
}
```
