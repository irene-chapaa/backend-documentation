# Register API

**Endpoint:**

```
POST {{Url}}/auth/register
```

**Authorization:** Bearer Token

```
{{Token}}
```

**Request Body**:

```json
{
  "firstName": "Ouda",
  "lastName": "Wycliffe",
  "email": "Wycliffe@gmail.com",
  "phone": "+254746764504",
  "password": "Ouda@1234"
}
```
