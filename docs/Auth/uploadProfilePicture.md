# Upload Profile Picture

**Endpoint:**

```
POST {{Url}}/auth/me/profilePicture
```

**Authorization:** Bearer Token

```
{{Token}}
```

**Request Body**:

```json
{
  "images": "
postman-cloud:///1eee2842-cd80-48a0-a43d-85db57ad46eb",
}
```
